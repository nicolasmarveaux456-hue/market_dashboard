# Market Dashboard

## Objective

This project transforms the Europe Momentum Backtest project into an interactive dashboard.

The objective is to provide a simple interface for exploring:

- market data
- strategy performance
- risk metrics
- portfolio composition
- robustness analysis

The dashboard is built using Streamlit and Plotly.

---

## Current Features

### Market Data Exploration

- Interactive ticker selection
- Historical price visualization
- Monthly returns visualization
- Plotly interactive charts
